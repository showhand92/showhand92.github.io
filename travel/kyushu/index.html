九州 3 晚 4 天行程地图

福冈进出，尽量少停留福冈；主线为熊本城、鹿儿岛城山展望台、樱岛、仙岩园。默认使用 OpenStreetMap 底图，联网时加载。

Day 1｜福冈 -> 熊本

  1. D1-110:55 福冈机场国际航站楼抵达。
  2. D1-212:25 地铁福冈机场站 -> 博多站。
  3. D1-313:01 SAKURA 751 博多 -> 熊本。
  4. D1-414:35 樱之马场城彩苑午餐/小吃。
  5. D1-515:10 熊本城。照片
  6. D1-618:30 下通/上通晚餐。

Day 2｜熊本 -> 鹿儿岛

  1. D2-109:10 水前寺成趣园，可选。照片
  2. D2-212:40 SAKURA 749 熊本 -> 鹿儿岛中央。
  3. D2-314:10 鹿儿岛中央/天文馆午餐。
  4. D2-415:45 城山展望台看樱岛。照片
  5. D2-518:30 天文馆晚餐。照片

Day 3｜樱岛 + 仙岩园

  1. D3-109:00 樱岛渡轮，鹿儿岛港 -> 樱岛港。
  2. D3-209:30 Sakurajima Island View 巴士环线。
  3. D3-310:30 汤之平展望所/熔岩步道。照片
  4. D3-411:40 渡轮返回鹿儿岛港。
  5. D3-512:45 仙岩园、尚古集成馆、咖啡。照片
  6. D3-618:00 鹿儿岛最后一晚晚餐。照片

Day 4｜鹿儿岛 -> 福冈机场

  1. D4-110:44 SAKURA 752 鹿儿岛中央 -> 博多。
  2. D4-212:30 博多站午餐/伴手礼。
  3. D4-313:30 Canal City/櫛田神社短逛，可选。照片
  4. D4-414:45 回博多站 -> 福冈机场，15:45 到达国际航站楼。
  5. D4-518:00 9C6538 福冈 -> 上海浦东。
